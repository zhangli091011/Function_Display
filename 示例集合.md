## 🎓 教学专用示例

### 9. 导数几何意义

#### 示例 9.1：切线斜率
```
函数：x^2
x = 2
f(2) = 4
f'(2) = 4
说明：在点 (2, 4) 处切线斜率为 4
```

#### 示例 9.2：极值点
```
函数：x^3 - 3*x
导数：3*x^2 - 3
令 f'(x) = 0：x = ±1
说明：x = 1 是极小值点，x = -1 是极大值点
```

### 10. 函数单调性

#### 示例 10.1：单调递增
```
函数：x^3
导数：3*x^2
说明：导数恒 ≥ 0，函数单调递增
```

#### 示例 10.2：单调性变化
```
函数：x^3 - 3*x
导数：3*x^2 - 3 = 3(x^2 - 1)
说明：
  x < -1: f'(x) > 0，递增
  -1 < x < 1: f'(x) < 0，递减
  x > 1: f'(x) > 0，递增
```

## 🎓 教学场景设计

### 场景 1：导数定义
```
1. 输入 f(x) = x^2
2. 计算 f(2) = 4, f(2.1) = 4.41
3. 斜率 ≈ (4.41 - 4) / 0.1 = 4.1
4. 求导：f'(2) = 4
5. 结论：导数是极限情况下的斜率
```

### 场景 2：链式法则
```
1. 输入 f(x) = sin(x^2)
2. 外层：sin(u), u' = cos(u)
3. 内层：u = x^2, u' = 2x
4. 结果：f'(x) = cos(x^2) * 2x
5. 验证：程序自动计算相同结果
```

### 场景 3：极值问题
```
1. 输入 f(x) = -x^2 + 4x
2. 求导：f'(x) = -2x + 4
3. 令 f'(x) = 0：x = 2
4. 计算 f(2) = 4
5. 结论：最大值在 (2, 4)
```
