## ⌨️ 输入语法

### 基本运算
```
加法：x + 2
```

## ⚠️ 注意事项

### 三角函数
- 使用**弧度制**
- `sin(pi) = 0`
- `cos(pi) = -1`

### 对数函数
- `log(x)` 要求 `x > 0`
- `log(a, x)` 要求 `a > 0` 且 `a ≠ 1`

### 除法
- 避免除以零
- `1/x` 在 `x=0` 时无定义

### 幂运算
- `2^3^2 = 2^(3^2) = 512`（右结合）
- 建议用括号明确优先级

## 🐛 常见错误

### "表达式解析错误"
- 检查括号是否匹配
- 检查函数名拼写

### "变量 x 未赋值"
- 数值计算需要输入 x 值
- 绘图不需要 x 值

### "除数不能为零"
- 检查表达式中的除法
- 避免 `1/0` 类型的错误

### "对数函数参数必须大于 0"
- `log(x)` 要求 `x > 0`
- 检查 x 的取值范围

## 🎓 教学技巧

### 演示函数性质
```
输入 x^2 和 -x^2，对比图像
观察对称性
```

### 理解导数
```
输入 x^2，绘制导函数
观察：导数 2x 表示切线斜率
```

### 探索周期性
```
输入 sin(x) 和 sin(2*x)
对比周期变化
```

### 复合函数
```
输入 sin(x^2)
求导：cos(x^2) * 2*x
理解链式法则
```
