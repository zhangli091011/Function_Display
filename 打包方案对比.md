# 打包方案对比

## 方案一：简易打包（推荐新手）

### 优点
- ✅ 操作简单，一键完成
- ✅ 不需要额外软件
- ✅ 生成单个 .exe 文件
- ✅ 可直接分发 ZIP 包

### 缺点
- ❌ 没有安装向导
- ❌ 不会创建开始菜单快捷方式
- ❌ 用户需要手动解压

### 使用方法
```bash
# 运行简易打包脚本
简易打包.bat

# 输出文件
dist\数学函数计算器.exe          # 可执行文件
数学函数计算器_v1.0.zip          # 分发包
```

### 适用场景
- 内部测试
- 小范围分发
- 快速原型
- 不需要专业安装程序

---

## 方案二：专业安装包（推荐正式发布）

### 优点
- ✅ 专业的安装向导
- ✅ 自动创建快捷方式
- ✅ 完整的卸载功能
- ✅ 用户体验更好

### 缺点
- ❌ 需要安装 Inno Setup
- ❌ 步骤稍多
- ❌ 需要学习基本配置

### 使用方法
```bash
# 步骤 1：打包可执行文件
一键打包.bat

# 步骤 2：安装 Inno Setup
# 下载地址: https://jrsoftware.org/isdl.php

# 步骤 3：编译安装脚本
# 右键点击 installer_script.iss -> Compile

# 输出文件
installer_output\数学函数计算器_安装程序_v1.0.exe
```

### 适用场景
- 正式发布
- 大范围分发
- 学校部署
- 需要专业形象

---

## 方案三：便携版（无需安装）

### 优点
- ✅ 无需安装
- ✅ 可放在 U 盘
- ✅ 不留注册表信息
- ✅ 适合教室电脑

### 缺点
- ❌ 文件较多
- ❌ 启动稍慢

### 使用方法
```bash
# 使用 --onedir 模式打包
pyinstaller --name=数学函数计算器 ^
            --windowed ^
            --onedir ^
            main.py

# 输出文件夹
dist\数学函数计算器\
    ├── 数学函数计算器.exe
    ├── _internal\          # 依赖文件
    └── ...
```

### 适用场景
- U 盘分发
- 临时使用
- 多台电脑快速部署
- 不允许安装软件的环境

## 推荐流程

### 开发阶段
1. 使用 `简易打包.bat` 快速测试
2. 验证功能是否正常

### 测试阶段
1. 使用 `简易打包.bat` 生成测试版
2. 分发给小范围用户测试

### 发布阶段
1. 使用 `一键打包.bat` + Inno Setup
2. 生成专业安装包
3. 同时提供便携版（可选）

## 常见问题

### Q1: 选择哪个方案？
**A:** 
- 新手/测试 → 简易打包
- 正式发布 → 专业安装包
- 教室使用 → 便携版

### Q2: 文件太大怎么办？
**A:** 
- 使用虚拟环境打包
- 排除不必要的模块
- 使用 UPX 压缩

### Q3: 打包后无法运行？
**A:** 
- 检查依赖是否完整
- 使用 `--debug=all` 查看错误
- 尝试 `--onedir` 模式

### Q4: 如何更新版本？
**A:** 
- 修改代码
- 更新版本号
- 重新打包
- 分发新版本

## 快速决策

**如果你：**
- 🎯 第一次打包 → 使用 `简易打包.bat`
- 🎯 需要专业形象 → 使用 `一键打包.bat` + Inno Setup
- 🎯 需要便携使用 → 修改为 `--onedir` 模式
- 🎯 不确定 → 先用简易打包测试
